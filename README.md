# Key Insights
- Technology category generated highest sales
- Some products had high sales but negative profit
- Consumer segment contributed highest revenue
- Certain regions underperformed in profitability

---

# Business Recommendations
- Focus on high-margin products
- Reduce loss-making inventory
- Improve customer retention
- Optimize pricing strategy
